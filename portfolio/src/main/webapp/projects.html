Python Projects

Hypermines

N-dimensional minesweeper. Probably not as good-looking as Microsoft's version, but I don't think you can play their version in a hypercube. Made for 6.009, MIT's Fundamentals of Programming course. Web app not available, but feel free to download the source code!

See More

To run the app from your computer, unzip the files, right click server.py and go to properties, and copy the location. Then, open up the terminal, and type "cd" and then paste the location of server.py. Then, type "python3 server.py", wait a little bit, and then type "localhost:8000" on your web browser.

NN-Allstar

More of an attempt to buy into the ML hype and learn some Tensorflow than a serious attempt at making a model, this neural net takes training data from the last 20 years and tries to predict who will be an All-Star in the current year. It was great to learn about and contribute to the only good basketball data scraper out there.

Julia Projects

COVID-19 Walkthrough

Final project for 6.S083, MIT's class called "Computational Thinking in Julia". I made a walkthrough of how to use gradient descent to fit real-world COVID data to a SEIR Model. Fitting this data is extremely important, as these models give us powerful insights about the spread of the disease, but finding the right parameters for the differential equations is not an easy task.
